# react-context-starter

A starter pack for React, Context API and PostCSS, all bundled using Webpack

## Instructions

Run the following command lines singly:

### Clone the repository

`git clone https://github.com/3tmaan/react-context-starter.git`

### Change directory to the repo

`cd react-context-starter`

### Install the app dependencies

`yarn` or `yarn install`

### Start the app in development mode

`yarn start`

### Build the app for production

`yarn start build`
